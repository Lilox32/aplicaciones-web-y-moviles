#Aplicaciones Web Moviles - 2do bachillerate informatica

Repositorio oficial de la asignatura **Aplicaciones Web Y Moviles**

---
## Descripcion
En esta asignatura los estuandiastes desarrollaran habilidades para el diseño y construccion de aplicaciones web
El enfoque de la materua esta basado en:

- Desarrollo Practico de software
- Construccion progresiva de aplicaciones reales
- Trabajo colaborativo y resolucion del problema

  ## Objetivos de la asignatura
  Al finalizar el curso, el estudiante sera capas de:
  - Crear interfaces funciones
  - Aplicar logica de programacion con JavaScript
  - Desarrollar aplicacion interactivas
  - Diseñar interfaces responsivas
  - Aplicar buenas practivas de desarrollo
